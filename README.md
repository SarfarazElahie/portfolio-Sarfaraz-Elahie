# Sarfaraz Elahie — Full Stack Developer Portfolio

A modern, responsive personal portfolio website built to showcase my skills, projects, and contact information as a Full Stack Web Developer.

---

## 🚀 Live Demo

[View Portfolio] : https://sarfaraz-elahie-portfolio-2026.netlify.app/

---

## 🛠️ Built With

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, animations, responsive design
- **Vanilla JavaScript** — Interactivity, scroll effects, form handling
- **Formspree** — Contact form backend
- **Google Fonts** — Inter & Poppins typefaces

> 🤖 This portfolio was built with the assistance of **Gemini AI**.

---

## ✨ Features

- 🌗 Light / Dark theme toggle with localStorage persistence
- 📱 Fully responsive — mobile, tablet, and desktop
- 🎨 Animated gradient background with floating orbs
- ⌨️ Typing effect on the code window
- 🃏 3D tilt effect on project cards
- 🔗 Active navigation link highlighting on scroll
- 📬 Contact form with real-time submission feedback via Formspree
- 🎞️ Scroll-triggered fade-in animations (custom Intersection Observer)
